summary: Ένα οπτικό εργαλείο που βοηθά τους εκπαιδευτικούς να σχεδιάζουν μαθήματα με το EntreComp, χρησιμοποιώντας έναν δομημένο καμβά και καθοδηγητικές κάρτες για τον σχεδιασμό, την αξιολόγηση και την παρακολούθηση των επιχειρηματικών ικανοτήτων.
description: Ο Καμβάς Ενσωμάτωσης EntreComp είναι ένα εργαλείο οπτικοποίησης που παρέχει μια ολιστική επισκόπηση των στοιχείων που απαιτούνται για τη δημιουργία ενός μαθήματος όπου περιλαμβάνονται επιχειρηματικές ικανότητες. Ο Καμβάς περιλαμβάνει ένα σύνολο καρτών για την καθοδήγηση των εκπαιδευτικών στη συμπλήρωση του καμβά δίνοντάς τους ερωτήσεις-εναύσματα για κάθε μέρος του καμβά. Στο τέλος, θα έχουν όλες τις απαραίτητες πληροφορίες και στοιχεία για την εφαρμογή και παρακολούθηση ενός μαθήματος.
outcomes: Αυξημένη ευαισθητοποίηση και κατανόηση των ικανοτήτων EntreComp. Ενσωμάτωση των ικανοτήτων EntreComp στα μαθήματα οποιουδήποτε πεδίου. Ανάπτυξη επιχειρηματικών ικανοτήτων μεταξύ των φοιτητών. Αξιολόγηση των επιχειρηματικών ικανοτήτων στην αρχή και στο τέλος ενός μαθήματος.
instructions: |
  1. Κατεβάστε τον Καμβά και εκτυπώστε τον εάν είναι απαραίτητο.
  2. Κατεβάστε τις Κάρτες και εκτυπώστε τες εάν είναι απαραίτητο.
  3. Χρησιμοποιήστε τις Κάρτες για να συμπληρώσετε κάθε μέρος του καμβά. Προσπαθήστε να συνεργαστείτε με άλλους συναδέλφους για να συζητήσετε τις ερωτήσεις και τις πληροφορίες που περιλαμβάνονται στις κάρτες.
  4. Σχεδιάστε το μάθημα σύμφωνα με τον συμπληρωμένο Καμβά.
