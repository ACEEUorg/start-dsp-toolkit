# German PDF Files

Place German translations of PDF files in this directory.

**File naming:** Use the same filenames as the English versions in `/en/` directory.

Example:
- `tool_1.1_TheEntrepreneurialWallPack.pdf`
- `tool_2.1_EntrepreneurialDictionary.pdf`
- etc.

**Note:** If a PDF is not available in German, the system will automatically fall back to the English version.
